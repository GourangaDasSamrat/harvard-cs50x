DNAnalyzer

A highly efficient, powerful, and feature-rich algorithm for analyzing DNA sequences

Introduction

DNAnalyzer identifies proteins, amino acids, start and stop codons, high coverage regions, regions susceptible to neurodevelopmental disorders, transcription factors, and regulatory elements. Researchers are working to extract valuable information from such software to better understand human health and disease. Currently, we have a Command-Line-Interface (CLI) and are working on a Graphical User Interface (GUI) that will enable physicians to quickly and more easily interact with the software.

Background

The human genome is composed of over 3 billion base pairs, making human analysis nearly impossible. Powerful computational and statistical methods are necessary to decode the functional information hidden in DNA sequences. Current analytical tools make it difficult for geneticists and physicians to acquire crucial information to better understand humans.

Features

Start and Stop Codons

Indicate the start and stop of a protein. A protein consists of one or more chains of amino acids encoded in a gene. 2

High Coverage Regions

Regions of DNA that code for proteins and have high GC-content. These regions often contain important genomic information. 3

Neurodevelopmental Disorders

Disorders caused by genetic mutations affecting brain and nervous system development, including autism and ADHD. 4

Transcription Factors

Proteins that bind to DNA and regulate gene expression. 6

FASTA Files (.fa & .fab)

A text-based format used to store DNA sequences and metadata. Supports multi-line and single-line FASTA files. 7

Command-Line-Interface

The Methionine CLI allows running DNAnalyzer services from the command-line and scripting sequences of commands.

Graphical User Interface

A cross-platform JavaFX application that performs the same functions as the CLI. Currently available GUI features:

  • Set name of DNA file to analyze
  • Set minimum and maximum number of reading frames
  • Run analysis
  • View results
